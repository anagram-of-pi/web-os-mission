Welcome

Welcome!

Hi, I'm Pi, and this is my WebOS app, made with pure HTML, CSS, and JS. You can learn more about this project in the About This Project window.

To navigate to other windows, find the Open option in the top menu and choose which to open. Alternatively, some links like this one will also open a window. Lastly, a few of the windows have interactive files on the desktop.


You can resize these windows by dragging the bottom right corner. Try it!


Close the window with the x in the top right corner of the window menu bar, or by pressing esc while the window is focused.

About This Project

About This Project

This project is a WebOS, albeit not a very good one. This was made for a mission as part of the Stardance program. The mission follows a guide to creating your own WebOS, but this has been heavily customized and has many features in addition to the base tutorial.


No AI was used to generate any code or text. AI was used to help debug and research.


Features

  • Moveable, resizable, closable windows
  • Window positions save on reload
  • Correct Z ordering after opening a page
  • Draggable files/apps on desktop
  • Double click menu bar to fullscreen window
  • Window cannot leave desktop
  • Window snaps back into frame
  • Multiple Apps
    • Text Documents (like this one!)
    • (Mostly) Functional Browser
    • Notes App
    • Settings App
  • Menu bar
  • Page Selector
  • Fullscreen OS
  • Browser/OS detection
  • Links that open windows in the OS
  • Keyboard shortcuts (esc, shift + esc, cmd + z)
  • Customizable themes through the settings app
